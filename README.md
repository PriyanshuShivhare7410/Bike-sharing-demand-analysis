# Bike-sharing-demand-analysis
Bike sharing demand analysis using Python to clean 10.8K+ records, perform statistical hypothesis testing, and generate actionable business insights.
### Tech Stack
* *Language:* Python
* *Libraries:* Pandas, NumPy, SciPy, Seaborn, Matplotlib

### Key Achievements
* *Data Cleaning & Engineering:* Cleaned and analyzed 10.8K+ bike rental records using Python libraries, engineered temporal features (hour, month), and removed multicollinearity by eliminating redundant variables.
* *Data Transformation:* Performed data transformation using logarithmic scaling (np.log) to address right-skewness and outliers, improving statistical validity of the analysis.
* *Hypothesis Testing:* Conducted statistical hypothesis testing using Welch's T-Test, Shapiro-Wilk Test, and Kruskal-Wallis Test to identify significant factors affecting bike rental demand.
* *Advanced Statistics:* Applied Chi-Square Test of Independence with contingency tables to evaluate the relationship between weather conditions and seasonal demand patterns.
* *Business Insights:* Generated actionable business insights, recommending dynamic pricing strategies during adverse weather and optimized fleet allocation to improve operational efficiency.
